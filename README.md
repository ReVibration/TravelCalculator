## TravelCalculator - UNIVERSITY PROJECT

This a calculator for employees in a fictional company that allow them to work out the claim for the journeys, mileage, engine size and whether the car was a company vehicle or privately owned.

# Problem it solves

It solves the problem of working out the amount the employee should claim back for their fuel based on their journey

# Key Features

Use Object-Oriented programing with different classes rather than procedural programming or event driven.
Uses a comand line interface 

# Tech Stack

Uses a Visual Studio template for comand line interfaces

# How to run it locally

To run this project locally you must download all of the files and open the SLN in visual studio

# Limitations 

Now this project was created for a university assignment therefore it has not been fully finshed or properly tested.
